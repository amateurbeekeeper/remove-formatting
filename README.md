# Remove Formatting

A simple Next.js tool to remove all formatting from text. Perfect for cleaning up copied text before pasting into emails.

## Features

- Automatically removes HTML formatting
- Strips rich text formatting
- Normalizes whitespace
- One-click copy to clipboard
- Clean, modern UI

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

1. Paste your formatted text into the input box
2. The plain text output will appear automatically
3. Click "Copy" to copy the plain text to your clipboard
4. Paste it into your email!

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
